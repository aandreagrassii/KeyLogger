# KeyLogger
KeyLogger in Python per intercettare su un file qualsiasi tasto schiacciato sulla tastiera.

Per evitare l'uso di questo tipologio di programma è consigliato usare 

Per eseguire il programma in background scaricare il file .pyw oppure eseguire su bash il comando: python3 keylogger.py &

Tutti i tasti verranno salvati sul file keyboard.txt
Modificare la stringa contenente il nome del file per poter cambiare nome al file, per modificare la posizione inserire il percorso dove è situato il file

E' necessario installare pynput:
pip3 install pynput
